# Asmi Verma Portfolio

A modern, responsive personal portfolio website focused on clean design, project storytelling, and professional presentation.

## Overview

This site is designed as a lightweight static portfolio with:

- Semantic HTML structure
- Dedicated CSS and JavaScript files for maintainability
- Responsive layout for desktop, tablet, and mobile
- Smooth scrolling, section reveal animations, and active navigation state

## Tech Stack

- HTML5
- CSS3 (custom properties, responsive grid, animations)
- Vanilla JavaScript (navigation, smooth scroll, intersection observer)
- Google Fonts (Playfair Display, Space Grotesk)

## Run Locally

1. Clone the repository.
2. Open the project folder in VS Code.
3. Open `index.html` in your browser.

You can also use VS Code Live Server for auto-reload while editing.

## Customization Guide

- Personal details: update text content in `index.html`.
- Colors and theme: edit CSS variables in `assets/css/styles.css` under `:root`.
- Motion and interactions: update behavior in `assets/js/main.js`.
- Images and resume: replace files in `assets/images/` and keep the same file names or update references in `index.html`.

## Deployment

This is a static site and can be deployed easily on:

- GitHub Pages
- Netlify
- Vercel

For GitHub Pages:

1. Push code to the `main` branch.
2. In repository settings, enable Pages and select the `main` branch root.
3. Save and open the generated URL.

## Accessibility and Performance Notes
